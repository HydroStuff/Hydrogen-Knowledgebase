Frequently Asked Questions

My script doesn't work or crashes

Not all scripts are compatible with Hydrogen. If a script doesn't work, it's either outdated or not compatible with Hydrogen. You will need to use a different script.

How to auto-execute

To auto-execute a script, open TextEdit, create a new document, and paste your script into it. Save it by pressing CMD + S and move it to the "autoexecute" folder within your Hydrogen folder. The file must be in .txt format, so you'll need to change its extension. In Finder, rename the file to end with .txt, for example, change "script.rtf" to "script.txt". When a pop-up asks if you want to use .txt, click "Use .txt".

How to remove Hydrogen

To remove Hydrogen from your Mac, you can use the script below.

cd && curl -LJs "https://raw.githubusercontent.com/nullstate9/roblox-stuff/refs/heads/main/executors/hydro-uninstaller.sh" -o "./hydro-uninstaller.sh" && chmod +x hydro-uninstaller.sh && sudo ./hydro-uninstaller.sh

Keep in mind that the script will keep the Hydrogen folder inside your home directory. It reinstalls the normal version of Roblox and removes the Hydrogen app. You can use it to play safely if Hydrogen is detected. If you want a full manual uninstall, delete both Roblox and Hydrogen, along with their folders, and then reinstall Roblox from the website.

Is my Mac supported

Both Intel and Silicon Macs are supported, as long as they are running macOS 11 or newer. You can check your version by typing sw_vers in the Terminal app.

Is Hydrogen Supported on Mobile or Windows devices

Currently, Hydrogen is supported on Mac devices only. The Android version has been discontinued, and there's no iOS or Windows version.
